# WZD Pinterest형 상단바 1차 기획 보고서

## 1. 이번 라운드 목표
Pinterest형 구조 전환의 첫 단계로, 상단을 보드 선택 중심 구조로 더 명확하게 정리한다.

### 한 문장 목표
**상단은 보드 선택을 위한 바처럼 보이게 만들고, 나머지 요소는 시각적으로 한 단계 뒤로 물린다.**

## 2. In scope
- 상단 레이아웃 구조 조정
- 보드 선택 영역 우선순위 강화
- 상단 보조 액션/정보의 시각적 약화 또는 위치 조정
- 관련 CSS 최소 수정

## 3. Out of scope
- 좌측 메뉴 전면 개편
- 모바일 하단 메뉴 완성
- 카드 시스템 전체 적용
- 테마 시스템 적용
- 데이터 로직 변경

## 4. 우선 변경 대상
- `src/App.tsx`
- `src/styles/main.css`

## 5. 완료 기준
1. 상단을 봤을 때 가장 먼저 보드 선택 구조가 읽힌다.
2. 모바일에서 보드 본문 가시성이 유지된다.
3. 데스크톱도 명백히 깨지지 않는다.
4. build 통과 후 commit/push.

## 6. 회귀 체크
- 모바일 보드 탭 보임
- 모바일 보드 내용 보임
- 제목/핵심 구조 완전 소실 없음
- 데스크톱 상단 구조 유지

## 최종 정리
이번 라운드는 Pinterest형 전환의 1차로, **상단을 보드 선택 전용 바처럼 보이게 만드는 정리 라운드**다.
