# WZD 상단 헤더 제거 + 보드 전환 전용 바 검증 보고서

## 1. 실제 문제
현재 PC 상단은 보드 전환 외의 검색, 액션, 제목, 상태 등 여러 요소를 함께 품고 있어 Pinterest형 콘텐츠 우선 구조와 맞지 않는다.

## 2. 이번 라운드 목표
- 상단 헤더 성격의 요소를 제거
- 상단에는 보드 전환만 남김
- 본문 콘텐츠가 더 빨리 시작되도록 정리

## 3. In scope
- 상단 헤더 제거
- 검색/액션/제목/상태 영역 제거 또는 비노출
- 보드 전환 탭만 남기는 구조 조정
- 관련 App/CSS 최소 수정

## 4. Out of scope
- 좌측 메뉴 추가 개편
- 카드 스타일 리디자인
- 모바일 하단 메뉴
- 테마 시스템 적용
- 데이터 로직 변경

## 5. 회귀 체크리스트
- 상단에 보드 전환만 남는가
- 보드 전환이 실제 동작하는가
- 본문 콘텐츠가 바로 보이는가
- 모바일이 명백히 깨지지 않는가
- build 통과

## 최종 판단
### PASS
이번 라운드는 `상단 헤더 제거 + 보드 전환 전용 바만 유지`라는 단일 목표로 진행하는 것이 적절하다.
