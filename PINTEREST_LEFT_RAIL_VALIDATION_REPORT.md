# WZD Pinterest형 좌측 아이콘 레일 검증 보고서

## 1. 실제 문제
현재 좌측 영역은 여전히 보드/섹션/패널 성격이 강해서 Pinterest의 얇은 아이콘 레일과 다르다.

## 2. 이번 라운드 목표
- 좌측을 얇은 Pinterest형 아이콘 레일로 정리
- 앱 내비게이션만 남기기
- 보드 전환은 상단 역할로 유지
- 중앙 콘텐츠가 더 넓게 읽히도록 만들기

## 3. In scope
- 좌측 사이드바 폭/구조/스타일 최소 재정리
- 아이콘 중심 메뉴화
- 관련 App/CSS 최소 수정

## 4. Out of scope
- 보드 전환 구조 개편
- 카드 리디자인
- 모바일 하단 메뉴
- 테마 작업
- 데이터 로직 변경

## 5. 회귀 체크리스트
- 좌측이 얇은 아이콘 레일처럼 보이는가
- 상단 보드 전환 구조가 유지되는가
- 중앙 콘텐츠 영역이 더 넓게 읽히는가
- 모바일이 명백히 깨지지 않는가
- build 통과

## 최종 판단
### PASS
이번 라운드는 `Pinterest형 얇은 좌측 아이콘 레일`만을 목표로 진행하는 것이 적절하다.
