# WZD 모바일 보드 가시성 복구 기획 보고서

## 1. 이번 라운드 목표
모바일에서 보드 탭은 보이지만 실제 보드 콘텐츠가 안 나오는 회귀 버그를 복구한다.

### 한 문장 목표
**모바일에서 현재 보드의 실제 콘텐츠 영역이 다시 확실히 보이도록 복구한다.**

## 2. In scope
- 모바일 보드 콘텐츠 영역 가시성 복구
- 상단 패널/컨테이너가 본문을 가리지 않도록 조정
- 관련 App 레이아웃 및 CSS 최소 수정

## 3. Out of scope
- 데스크톱 레이아웃 개선
- 추가 기능 개발
- 랜딩/공모전 메시지 작업
- 디자인 감성 고도화

## 4. 우선 변경 대상
- `src/App.tsx`
- `src/styles/main.css`

## 5. 개발자 태스크
- 모바일에서 보드 본문이 실제로 보이지 않는 원인을 찾고 복구
- 보드 탭 전환 후에도 본문이 가려지지 않게 수정
- 상단 구조는 유지하되 본문 우선순위를 되찾게 조정
- build 통과 후 commit/push

## 6. 완료 기준
1. 모바일에서 보드 칩 아래 실제 보드 내용이 보인다.
2. 옆으로 슬라이드/탭 변경 후에도 본문이 유지된다.
3. 상단 구조와 본문이 겹치거나 가려지지 않는다.
4. build가 통과한다.

## 최종 정리
이번 라운드는 미세 개선이 아니라 **회귀 버그 복구**다. 다른 작업보다 우선해서 바로 마감해야 한다.
