# WZD Pinterest형 좌측 아이콘 레일 기획 보고서

## 1. 이번 라운드 목표
좌측을 얇은 Pinterest형 아이콘 레일로 단순화해 앱 메뉴 역할만 하게 만든다.

### 한 문장 목표
**좌측은 얇은 아이콘 메뉴, 상단은 보드 전환, 중앙은 콘텐츠라는 구조를 더 분명히 만든다.**

## 2. In scope
- 좌측 폭 축소
- 아이콘 중심 메뉴 구조 정리
- 텍스트/섹션/과한 장식 최소화
- 관련 CSS 최소 수정

## 3. Out of scope
- 보드 전환 위치 변경
- 카드/위젯 리디자인
- 모바일 하단 메뉴
- 테마 작업
- 데이터 로직 변경

## 4. 우선 변경 대상
- `src/App.tsx`
- `src/styles/main.css`

## 5. 완료 기준
1. 좌측이 얇은 아이콘 레일처럼 보인다.
2. 상단 보드 전환은 유지된다.
3. 중앙 콘텐츠 영역이 더 넓고 단순하게 읽힌다.
4. build 통과 후 commit/push.

## 최종 정리
이번 라운드는 Pinterest형 전환의 데스크톱 핵심 라운드로, **좌측을 얇은 앱 메뉴 레일로 바꾸는 단일 목적 작업**이다.
