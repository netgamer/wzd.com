# WZD 모바일 상단영역 재검수 기획 보고서

## 1. 이번 라운드 목표
모바일 상단영역에서 잃어버린 제목/메뉴 인지를 복구하면서, 과한 상단 높이와 공백은 줄인다.

### 한 문장 목표
**제목과 메뉴는 다시 선명하게 살리고, 쓸데없는 상단 공백만 줄여 모바일 첫인상을 안정적으로 만든다.**

## 2. In scope
- 모바일 상단 제목 가시성 복구
- 메뉴/구조 버튼 가시성 복구
- 큰 상단 패널 높이 축소
- 칩/탭 영역 슬림화
- 관련 CSS 최소 수정

## 3. Out of scope
- 데스크톱 헤더 재설계
- 기능 추가
- 전체 보드 레이아웃 개편
- 저장 로직 수정

## 4. 우선 변경 대상
- `src/App.tsx`
- `src/styles/main.css`

## 5. 개발자 태스크
- 모바일에서 현재 보고 있는 보드 제목이 첫 시야에서 분명히 읽히게 조정
- 메뉴 버튼/구조 버튼이 사라진 것처럼 보이지 않게 복구
- 상단 큰 공백 영역을 줄여 실제 콘텐츠가 더 빨리 보이게 수정
- build 통과 후 commit/push

## 6. 완료 기준
1. 모바일 스크린샷에서 제목이 명확히 보인다.
2. 메뉴/구조 버튼이 다시 인지된다.
3. 상단 전체 높이는 줄어든다.
4. 사용자가 더 빨리 실제 콘텐츠로 진입할 수 있다.
5. build가 통과한다.

## 최종 정리
이번 라운드는 `더 줄이는 라운드`가 아니라, **중요한 것을 다시 살리면서 불필요한 높이만 덜어내는 정리 라운드**다.
