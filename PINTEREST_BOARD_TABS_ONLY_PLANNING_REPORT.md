# WZD 상단 헤더 제거 + 보드 전환 전용 바 기획 보고서

## 1. 이번 라운드 목표
상단 헤더를 없애고, 보드 전환만 남는 Pinterest형 상단 구조로 단순화한다.

### 한 문장 목표
**상단은 보드 전환 탭만 남기고, 나머지 헤더성 요소는 걷어낸다.**

## 2. In scope
- 상단 헤더 제거
- 보드 전환 전용 바 유지
- 본문이 더 빨리 시작되도록 레이아웃 정리
- 관련 CSS 최소 수정

## 3. Out of scope
- 좌측 메뉴 추가 수정
- 카드/위젯 전체 디자인 변경
- 모바일 하단 메뉴
- 테마 작업
- 데이터 로직 변경

## 4. 우선 변경 대상
- `src/App.tsx`
- `src/styles/main.css`

## 5. 완료 기준
1. 상단에 보드 전환만 남는다.
2. 본문 콘텐츠가 바로 보인다.
3. 보드 전환은 유지된다.
4. build 통과 후 commit/push.

## 최종 정리
이번 라운드는 Pinterest형 전환의 핵심 라운드로, **헤더성 상단을 제거하고 보드 전환만 남기는 단일 목적 작업**이다.
